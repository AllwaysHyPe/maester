# Maester - Out of the box tests

## Overview
This page lists the collection of community contributed tests that you can use to get started with Maester.

* Maester Conditional Access Policy Tests: These tests validate if your conditional access policies are following generally recommended best practices.
* Azure AD Security Config Analyzer Tests: These tests validate tenant security config and are based on the [Azure AD Attack and Defense - Security Config Analyzer](AADSC)

See the [Getting started](/docs/intro) page on how to run these tests.
