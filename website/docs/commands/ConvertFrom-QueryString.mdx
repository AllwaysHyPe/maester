## SYNOPSIS

Convert Query String to object.

## SYNTAX

```powershell
ConvertFrom-QueryString [-InputStrings] <String[]> [-DecodeParameterNames] [-AsHashtable]
 [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION

\{\{ Fill in the Description \}\}

## EXAMPLES

### EXAMPLE 1

```powershell
ConvertFrom-QueryString '?name=path/file.json&index=10'
Convert query string to object.
```

### EXAMPLE 2

```powershell
'name=path/file.json&index=10' | ConvertFrom-QueryString -AsHashtable
Convert query string to hashtable.
```
