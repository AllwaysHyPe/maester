## SYNOPSIS

Resets the local cache of DNS lookups.
Use this if you need to force a refresh of the cache in the current session.

## SYNTAX

```powershell
Clear-MtDnsCache
```

## DESCRIPTION

By default all DNS responses are cached and re-used for the duration of the session.

Use this function to clear the cache and force a refresh of the data.

## EXAMPLES

### EXAMPLE 1

```powershell
Clear-MtDnsCache
```

This example clears the cache of all DNS lookups.
