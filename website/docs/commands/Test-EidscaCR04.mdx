## SYNOPSIS

Checks if Consent Framework - Admin Consent Request - Consent request duration (days)???
is set to '30'

## SYNTAX

```powershell
Test-EidscaCR04 [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION

Specifies the duration the request is active before it automatically expires if no decision is applied

Queries policies/adminConsentRequestPolicy
and returns the result of
 graph/policies/adminConsentRequestPolicy.requestDurationInDays -eq '30'

## EXAMPLES

### EXAMPLE 1

```powershell
Test-EidscaCR04
```

Returns the result of graph.microsoft.com/beta/policies/adminConsentRequestPolicy.requestDurationInDays -eq '30'
