## SYNOPSIS

Helper method to connect to Microsoft Graph using Connect-MgGraph with the required scopes.

## SYNTAX

```powershell
Connect-Maester [-SendMail] [-UseDeviceCode] [-Graph] [-Environment {China | Germany | Global | USGov | USGovDoD}] [<CommonParameters>]

Connect-Maester [-SendMail] [-UseDeviceCode] [-All] [-Environment {China | Germany | Global | USGov | USGovDoD}] [-AzureEnvironment {AzureChinaCloud | AzureCloud | AzureUSGovernment}] [<CommonParameters>]

Connect-Maester [-SendMail] [-UseDeviceCode] [-Azure] [-AzureEnvironment {AzureChinaCloud | AzureCloud | AzureUSGovernment}] [<CommonParameters>]
```

## DESCRIPTION

Use this cmdlet to connect to Microsoft Graph using Connect-MgGraph.

This command is completely optional if you are already connected to Microsoft Graph using Connect-MgGraph with the required scopes.
```
Connect-MgGraph -Scopes (Get-MtGraphScope)
```

Additional API connections beyond Graph are supported with switches. Such as Azure using Connect-AzAccount.

## EXAMPLES

### EXAMPLE 1

```powershell
Connect-Maester
```

### EXAMPLE 2

```powershell
Connect-Maester -All
```
