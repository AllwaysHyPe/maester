## SYNOPSIS

Get object property value.

## SYNTAX

```powershell
Get-ObjectProperty [-InputObjects] <Object> -Property <String[]> [-ProgressAction <ActionPreference>]
 [<CommonParameters>]
```

## DESCRIPTION

\{\{ Fill in the Description \}\}

## EXAMPLES

### EXAMPLE 1

```powershell
$object = New-Object psobject -Property @{ title = 'title value' }
PS C:\>$object | Get-ObjectProperty -Property 'title'
Get value of object property named title.
```

### EXAMPLE 2

```powershell
$object = New-Object psobject -Property @{ lvl1 = (New-Object psobject -Property @{ nextLevel = 'lvl2 data' }) }
PS C:\>Get-ObjectProperty $object -Property 'lvl1', 'nextLevel'
Get value of nested object property named nextLevel.
```
