## SYNOPSIS

Returns the list of Graph scopes required to run Maester.

## SYNTAX

```powershell
Get-MtGraphScope
```

## DESCRIPTION

Use this cmdlet to connect to Microsoft Graph using Connect-MgGraph.

## EXAMPLES

### EXAMPLE 1

```powershell
Connect-MgGraph -Scopes (Get-MtGraphScope)
```
